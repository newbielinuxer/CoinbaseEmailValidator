# CoinbaseEmailValidator
 Check Is Email Registered in Coinbase.com
 
Install
---------
```python
pip install -r requirments.txt
```
Usage
---------

```python

USE_PROXY = False # True / False
PROXY_HOST = '131.153.133.18'  # rotating proxy
PROXY_PORT = 6011
PROXY_USER = 'USERNAME'
PROXY_PASS = 'PASSWORD

python coinbase.py
```
